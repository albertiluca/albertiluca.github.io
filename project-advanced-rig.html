Advanced Human Rig

A Technical Art project focused on creating a modular and scalable rigging system for humanoid characters.

Gallery

Project Goals

The primary goal was to build a feature-rich humanoid rig from scratch, designed to provide animators with an efficient and intuitive workflow.

Role

Rigger & Technical Artist

Key Technical Features

  • IK/FK Switch and Snap: Full IK/FK switching for arms and legs with seamless snapping between modes.
  • Flexible Spline IK Spine: A robust spline-based IK spine allowing for natural twisting and bending.
  • Advanced Foot Roll System: Includes controls for foot roll, side-to-side tilt, and toe lift for realistic foot placement.
  • Scalable and Modular Structure: A clean hierarchy and Python scripting to automate repetitive tasks and ensure scalability.

Tools Used:

  • Autodesk Maya
  • Python (for scripting)
